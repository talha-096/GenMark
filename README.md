# GenMark

AI-powered marketing content generation platform.

## Monorepo Structure

```
GenMark/
├── Frontend/          # React + Vite + TypeScript SPA
└── Backend/           # Python FastAPI backend
```

## Quick Start

### Frontend
```bash
cd Frontend
npm install
npm run dev
```

### Backend
```bash
cd Backend
pip install -r requirements.txt
uvicorn main:app --reload
```
